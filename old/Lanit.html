Привет, искатель приключений!

Если ты хочешь установить программу, то тебе необходимо выполнить следующие условия:

  1. Скачать программу MRS
  2. Поставить SQL Server 2016 на одну машину
  3. Создать базу данных
  4. Запустить скрипт для нужной базы данных этой машине
  5. Заставить программиста создать сервер KOMPOSQLSERVER
  6. Заставить программиста загрузить базу на созданный сервер
  7. Welcome to the Matirx
